## Installation

<h3>Clone the Repository:</h3>

```
git clone https://github.com/dineshdevelope/Samantha-Biography-Page.git
```

Navigate to the project

```
cd php-course-landing-page
```

Install the Dependencies

```
npm install
```

Run the Project

```
npm run dev
```
